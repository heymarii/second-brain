# 🧠 Second Brain

> A living, breathing knowledge base — updated daily. Every day I learn something new and commit it here.

## Structure

```
second-brain/
├── til/          ← Today I Learned (daily notes)
├── resources/    ← Curated links, tools, and references
├── learning/     ← Structured roadmaps and course notes
└── templates/    ← Reusable Markdown templates
```

## How to Use This Repo

- **`til/`** — Each file is a daily entry. Browse to find quick answers and concepts.
- **`resources/`** — Categorized lists of the best tools, articles, and references I've found.
- **`learning/`** — Deeper structured notes on topics I'm actively studying.
- **`templates/`** — Copy-paste templates I use regularly.
