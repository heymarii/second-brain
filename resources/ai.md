# AI & Machine Learning Resources

*Last updated: 2026-03-27*

## Foundational Reading

- [Attention Is All You Need](https://arxiv.org/abs/1706.03762) — The original transformer paper
- [The Illustrated Transformer](https://jalammar.github.io/illustrated-transformer/) — Best visual explainer
- [Andrej Karpathy's Neural Networks Zero to Hero](https://karpathy.ai/zero-to-hero.html) — Best free course

## Prompt Engineering

- [Anthropic Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview)
- [dair-ai/Prompt-Engineering-Guide](https://github.com/dair-ai/Prompt-Engineering-Guide) — Comprehensive GitHub repo
- [Learn Prompting](https://learnprompting.org/) — Free interactive course

## Agent Frameworks

- [LangChain](https://python.langchain.com/) — Most popular, large ecosystem
- [LlamaIndex](https://www.llamaindex.ai/) — Best for RAG and document pipelines
- [CrewAI](https://crewai.com/) — Multi-agent collaboration
- [Anthropic Claude API](https://docs.anthropic.com/) — Direct API, best for custom agents
